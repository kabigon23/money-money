# Asset-Master

Asset-Master는 개인 자산 관리 웹 애플리케이션입니다.

## 핵심 목표
미 증시와 국내 상장 미국 ETF 시세를 연동하여, 사용자가 설정한 태그별로 자산 비중을 시각화하여 보여주는 것을 핵심 목표로 합니다.

## 주요 기능
- **실시간/지연 시세 연동**: 미 증시 및 국내 상장 미국 ETF 가격 정보 반영
- **태그 기반 자산 분류**: 개별 자산에 커스텀 태그를 부여하여 관리
- **자산 비중 시각화**: 태그별 자산 비중 및 포트폴리오 분석 제공

## 기술 스택
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Language**: TypeScript
