# 💰 MoneyMoney

> 내 자산을 한눈에 — 실시간 시세 기반 포트폴리오 트래킹 앱

## 주요 기능

### 📊 자산 관리
- **종목 추가/수정/삭제** — 미국 주식, 국내 ETF, 암호화폐 지원
- **실시간/지연 시세 연동** — 미 증시 및 국내 상장 ETF 가격 자동 반영
- **KRW / USD 전환** — 환율 기반 실시간 평가금액 계산
- **거래 내역 기록** — 매수·매도·수동 수정 이력 자동 저장

### 🏷️ 분류 시스템
- **카테고리** — 액티브, 연금저축 등 원하는 이름으로 자산 묶기
- **태그** — 커스텀 색상 태그로 세분화된 분류
- **태그별 자산 비중 원그래프** — 포트폴리오 비중 시각화 (소수점 1자리 퍼센티지)

### 👤 멀티 유저
- **계정별 독립 데이터** — 사용자마다 자산/카테고리/태그 완전 분리
- **관리자 기능** — 계정 생성·수정·삭제, 비밀번호 초기화
- **쿠키 기반 세션** — 어느 기기에서나 로그인 유지 (7일)

### 📱 모바일 최적화
- 갤럭시·아이폰 등 모바일 화면에 맞춘 반응형 UI
- 모바일 전용 자산 카드 뷰

## 기술 스택

| 분류 | 기술 |
|---|---|
| Framework | Next.js 15 (App Router) |
| Language | TypeScript |
| Styling | Tailwind CSS |
| UI | shadcn/ui |
| Database | Redis (Upstash / Redis Cloud) |
| Auth | bcryptjs + HttpOnly Cookie Session |
| Deploy | Vercel |
